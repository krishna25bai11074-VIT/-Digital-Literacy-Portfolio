# -Digital-Literacy-Portfolio


Digital Literacy & Professional Identity
B.Tech Computer Science & Engineering | VIT Bhopal University
This repository contains the documentation and deliverables for my Digital Literacy course. It outlines the foundational steps I have taken to build a professional digital footprint, understand cybersecurity, and master essential digital tools.

🚀 Project Overview
The project is divided into five core tasks focusing on visual communication, professional networking, technical practice, digital ethics, and cybersecurity.

📋 Task Highlights
🎨 Task 1: Visual Communication
Asset: One-page infographic titled "Digital Literacy — Your Superpower in the Digital Age".

Tool: Canva.

Focus: Defining digital literacy, essential tools, safe internet practices, and professional email etiquette.

💼 Task 2: Digital Portfolio Setup
Established a professional presence across three major platforms to track my academic and technical growth:

GitHub: Primary repository for code and B.Tech projects.

LinkedIn: Professional networking and career updates (Expected Graduation: 2028).

Kaggle: Data Science and AI skill development.

💻 Task 3: Technical Platforms
Coding: Completed the "Solve Me First" challenge on HackerRank to familiarize myself with competitive programming environments.

Collaboration: Developed a Google Forms awareness quiz with automated response tracking via Google Sheets.

⚖️ Task 4: Digital Ethics & Footprint
Case Study: Analyzed a real-world incident regarding the permanence of social media posts.

Key Takeaway: Digital interactions are permanent and public; professional conduct is mandatory across all platforms.

🛡️ Task 5: Cybersecurity Awareness
Research: Focused on UPI and social engineering fraud in India.

Personal Security Protocol: Implemented a "Zero Trust" policy for unsolicited calls and saved the National Cyber Crime Helpline (1930).

🛠️ Tools & Technologies Used
Design: Canva

Version Control: Git & GitHub

Productivity: Google Workspace (Docs, Forms, Sheets)

Technical: HackerRank, LinkedIn, Kaggle

✍️ Reflection
The most significant takeaway from this course was the realization that a professional digital identity is built incrementally. Starting these profiles in my first semester allows me to document my journey from a beginner to a proficient engineer over the next four years.
